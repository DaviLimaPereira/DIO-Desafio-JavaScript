palindromo Verifique, de duas maneiras diferentes entre si, se uma String é um palíndromo. Palíndromo: frase ou palavra que se pode ler, indiferentemente, da esquerda para a direita ou vice-versa (ex.: raiar, ama, ovo, radar)
substituir números pares Troque todos os elementos pares e diferentes de zero de um array pelo número 0. Se o array for vazio, retorne -1. Exemplo: Input -> [1, 3, 4, 6, 80, 33, 23, 90] Output -> [1, 3, 0, 0, 0, 33, 23, 0] Input -> [] Output -> -1
